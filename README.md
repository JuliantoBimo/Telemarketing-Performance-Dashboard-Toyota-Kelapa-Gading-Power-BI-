## Tentang Project

Project ini merupakan dashboard monitoring telemarketing yang saya kembangkan dan implementasikan di **Toyota Kelapa Gading** sebagai bagian dari upaya meningkatkan kinerja after-sales melalui sistem monitoring berbasis data.

## Goal Setting â€” Metode SMART

Saya menggunakan pendekatan **SMART Goal Framework** agar target mudah diterapkan dan diukur.

| Parameter | Penjelasan |
|----------|------------|
| **Specific** | Meningkatkan kontribusi telemarketing pada jumlah unit entry servis |
| **Measurable** | Diukur berdasarkan booking rate, connected rate, dan unit entry |
| **Achievable** | Disesuaikan dengan kapasitas dan SOP call telemarketing |
| **Relevant** | Berdampak langsung pada revenue after-sales dan retention pelanggan |
| **Time-Based** | Monitoring harian, evaluasi mingguan & bulanan |
